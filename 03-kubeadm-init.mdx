## ğŸ›  Configure kubectl Access

Now copy the `admin.conf` to your user's `.kube` directory:

```bash
mkdir -p $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config
```

Once done, test access using:

```bash
kubectl get nodes
```

You should see the control plane node in a `NotReady` stateâ€”this is expected because the pod network is not yet installed.
